### Clientside
Config files for the mod can be found within your minecraft profile under `/config/jsg/jsgConfig_2.0`<br/>
When in single player, settings marked SERVER or CLIENT will be used. When playing on a server, only settings marked CLIENT will be used.

### Serverside
Config files for the mod can be found server root directory under `/config/jsg/jsgConfig_2.0`<br/>
Only settings marked SERVER will be used
