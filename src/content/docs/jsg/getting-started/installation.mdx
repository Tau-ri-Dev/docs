## With Modrinth
[Modrinth](https://modrinth.com/) is a popular Minecraft Profile manager that provides easy installing of mods, modloaders, resource packs, data packs, and mod packs.

To get started, download the latest client from [https://modrinth.com/app](https://modrinth.com/app)

To create a new profile with Just Stargate Mod
- Click the `+` button in the side bar
- Enter a name for the profile
- Select `Forge` or `Neoforge` as the loader
- Select game version `1.20.1` - NOTE: JSG is **not** compatible with other versions of 1.20
- Click create

### Allocating more RAM
If Minecraft crashes during startup, it might be due to not enough memory being dedicated.<br/>
To allocate more RAM, click the gear icon next to the play button, select "Java and Memory", and adjust the Memory Allocated.

1024MB is recommended when using JSG Mod on it's own<br/>
8192 MB is recommended if you're using it as part of a modpack

## With CurseForge
[CurseForge](https://curseforge.com/) is another popular Minecraft Profile manager that provides easy installing of mods, modloaders, resource packs, data packs, and mod packs.

To get started, download the latest client from [https://www.curseforge.com/download/app](https://www.curseforge.com/download/app)

To create a new profile with Just Stargate Mod
- Click the `+Create` button in the top bar
- Enter a name for the profile
- Select game version `1.20.1` - NOTE: JSG is **not** compatible with other versions of 1.20
- Select game type `Forge` or `Neoforge` as the loader
- Select newest Modloader Version 
- Click create

### Allocating more RAM
If Minecraft crashes during startup, it might be due to not enough memory being dedicated.<br/>
To allocate more RAM, open the modpack profile, click on the 3 dots next to the play button and select Profile Options.<br/>
Check **Custom RAM Allocation** box and adjust the amount of RAM allocated for this modpack.

1024MB is recommended when using JSG Mod on it's own<br/>
8192 MB is recommended if you're using it as part of a modpack

## Manual installation
The Microsoft launcher is **not capable of running mods itself**.<br/>
To get started, download the latest Minecraft Forge from [the Forge website](https://files.minecraftforge.net/net/minecraftforge/forge/index_1.20.1.html)

### Forge installation
- On this page, select the version of Minecraft for which you want to download Forge
- Download Installer - latest version. After downloading, open this installation file.
- If you have trouble opening the file, check if you have the latest version of [Java](https://www.java.com/en/download/) installed
- In the installation wizard, make sure the `Install Client` option is selected
- Check the path of the destination directory and press the `OK` button
- In a few moments, you will receive a message about the successful installation.
- Download the mods you want and place them in the target directory in the mods folder
- You should now be able to select your modified profile in Microsoft Launcher

If your new profile is not displayed, click on `Installations` in the top bar and check the Modded option

### Allocating more RAM
If Minecraft crashes during startup, it might be due to not enough memory being dedicated.<br/>
To allocate more RAM, click on `Installations` in the top bar.<br/>
Click on the installation you want to edit and in the newly opened window click on `more options`<br/>
and find the `JVM Arguments` field.<br/>
Find the flag `-XmX`<br/>
To allocate more RAM, adjust this flag, eg.: to allocate 4GB - `-Xmx4G`

1024 MB is recommended when using JSG Mod on it's own<br/>
8192 MB is recommended if you're using it as part of a modpack
