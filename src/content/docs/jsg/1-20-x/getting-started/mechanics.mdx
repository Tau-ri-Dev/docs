## Crystals
Stargate technology is based on crystal technology. Therefore, in addition to basic materials such as iron, copper, redstone, naquadah, you will also need crystals. These crystals can be obtained in several ways:

- By finding and mining a geode that contains crystal buds, clusters, crystal blocks, and budding blocks
- By finding a free-standing budding block on which buds and clusters grow
- By growing crystals using budding blocks

You can obtain a random number and size of crystals after mining certain crystal development stages:

- Small Bud: 1x small crystal
- Medium Bud: 1-2x small crystal 
- Large Bud: 0-1x crystal, 2x small crystal
- Crystal Cluster: 1x crystal, 2-3x small crystal

The resulting number of dropped crystals can be increased if you use a pickaxe with the Fortune enchantment!
You can also get these development stages if you use a pickaxe with the Silk Touch enchantment!

Each type of crystal listed above generates in a specific set of biomes. Each set has its own biome tag:
- Blue crystal: `#has_crystals_blue`
- Ender crystal: `#has_crystals_ender`
- Pegasus crystal: `#has_crystals_pegasus`
- Red crystal: `#has_crystals_red`
- Yellow crystal: `#has_crystals_yellow`
- White crystal: `#has_crystals_white`

Biomes specified in these tags by default can be found at: [Configuration/tags](/jsg/1-20-x/configuration/tags/#biome-tags)
## Budding blocks
Unlike Amethyst budding blocks, which do not drop anything when mined, JSG budding blocks drop crystal seeds and a 50% chance of basic material 
(the block from which the crystal seed crystallised and transformed into a budding block).
