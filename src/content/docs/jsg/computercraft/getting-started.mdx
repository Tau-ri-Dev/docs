ComputerCraft: Tweaked is a mod for Minecraft which adds programmable computers, turtles and more to the game. A fork of the much-beloved ComputerCraft, it continues its legacy with improved performance and stability, along with a wealth of new features.

![](https://raw.githubusercontent.com/Fredyman95/Repository-of-knowledge/refs/heads/main/JSG%20website/cc_integration/introduction.webp)

## Instalation
Mod is installed the same way as Just Stargate Mod - more info in the section [General/installation](/jsg/getting-started/installation/). 
Only exception is that developers no longer release new updates to platform [CurseForge](https://www.curseforge.com/minecraft/mc-mods/cc-tweaked) - to get the latest version of ComputerCraft it is necessary to use the platform [Modrinth](https://modrinth.com/mod/cc-tweaked).

## Official documentation
Official documentation with all the information about ComputerCraft: Tweaked mod can be found at: [https://tweaked.cc/](https://tweaked.cc/)
