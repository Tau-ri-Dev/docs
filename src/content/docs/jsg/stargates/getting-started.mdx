A stargate is a teleportation device created by the Ancients to allow them to easily and almost instantly travel not only within and 
between the Galaxy, but even across the universe. Each gate has its own set of symbols for a given region (galaxy).

<div style="width: 25dvw">
![Stargate](/assets/jsg/stargates/gate_milkyway.webp)
</div>

Since there are no galaxies or distant planets in Minecraft, the above system is modified so that stargates can be used in the game. 
Therefore, multiple stargates can be placed in 1 dimension and gates can also travel between dimensions. 
More about traveling between dimensions in [Dimensional Groups](/).

JSG adds stargates, DHDs and devices that were seen in the SG-1, SGA, SGU series and movies. 
The mod focuses on faithfully handling the models and functional mechanics e.g. Entering the address on the DHD by pressing physical buttons.

The mod contains 3 types of address systems. All symbols including all information can be found in [Stargates/Glyphs section](/jsg/stargates/glyphs)

### Selecting the correct type of address system:
Each stargate has these 3 types of addresses - these addresses are used to create a connection between stargates.<br/>
The required address type must be selected according to the type of stargate from which the address will be entered.<br/>
Therefore, it is not possible to create an outgoing connection from a Pegasus stargate using the MilkyWay or Universe address systems.

### Stargate address assignment
Each Stargate address is unique. There are several parameters used to create the address: The x,y,z coordinates and dimension in which the stargate base block is located.<br/>
Since the address is bound according to the parameters listed above, the Stargate address can be preserved in several ways:
- By breaking the old stargate and building a new one in its place
- Changing Stargate address using [command](/jsg/getting-started/commands/#-sgsetaddress)

Another welcome feature is that multiple stargates can be placed in one chunk

### Address length
Stargate addresses consist of 6, 7 or 8 symbols + Point of Origin *(PoO) (from here on will be listed with PoO included in the count)* 
and its minimum length is affected by the following parameters:
- Types of stargate address system that dials the address and destination stargate
- Dimension / Dimension group containing stargate that enters the address and the destination stargate
- PoO is always the last glyph in the address but may not always be displayed

## Construction
Stargates are multi-block structures that consist of a Base Block, Ring Blocks, and Chevron Blocks. The only exception is the Orlin's gate, which consists of only a Base Block and Member Blocks.

After placing the Base block, stargate can be built in several ways:
!- By placing the appropriate blocks in the correct position
!- By clicking the appropriate blocks on the Base Block - blocks will automatically place themselves in the correct position

The size, shape and number of blocks depend on the set gate size in [Stargate config][sg-config]
|  |  |
|--|--|
| ![](/assets/jsg/stargates/builder_classic_all.webp) | ![](/assets/jsg/stargates/builder_orlin.webp) |

Once all the blocks have been placed, the multi-block structure will automatically connect to form a stargate. You will know this by blocks disappearing and a functional stargate being placed in their place.
