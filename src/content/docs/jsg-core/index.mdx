JSG: Core contains utilities and core stuff to run all other JSG addons on it.

## Expanding JSG
JSG can be expanded with official add-ons. 
Currently available are add-ons introducing content from Stargate Universe, decorative blocks, and Transport Rings. 
The Transport Rings have undergone significant improvements, making them a truly viable short-range teleportation system.

import { CardGrid } from '@astrojs/starlight/components';
import JSGAddons from "/src/components/JSGAddons.astro";
import LinkCard from "/src/components/LinkCard.astro";
