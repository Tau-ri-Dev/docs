## What can I do with it?
- Create own address system and hook into notebook pages and cartouches
- Handle Force Chunkloading properly
- Configure BlockEntities in the game with our in-game config API
- Use our Raycaster API to create immersive experience for players
- Hook into CC: Tweaked or OpenComputers 2 without need to handle garbage stuff around
- Load and use custom sized textures (**NOW ALSO .webp!**) and .obj models
- Use tons of helper classes that are available inside this dependency
- and so much more...
